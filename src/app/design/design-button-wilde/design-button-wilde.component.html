Button & Wilde

Background

The client has a B2C ecommerce platform which sells personalised beauty products. Through remote moderated research, I discovered customers had difficulties using the online personalisation tool which led to a low conversion rate.

My Role

I proposed and lead the redesign process of the online personalisation tool. I restructured the layout and content on the homepage so the product concept is clearly communicated. I created a prototype with simplified user journey using Axure RP and conducted guerrilla user testing to refine the prototype.

My Contribution

  • Remote Moderated Research
  • Design Studio Workshop
  • Wireframing
  • Prototyping
  • Guerrilla User Testing
