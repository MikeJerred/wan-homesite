Guerrilla user testing

I went to a local cafe to conduct Guerrilla user testing to validate both versions of the wireframe, for each testing session a video was recorded using a screen capture tool with the participant's consent.

Guerrilla user testing

Most participants were able to understand the product concept quickly through the wireframes for the homepage. There was a mixed response to sections such as the live Instagram feed and featured products. Participants were able to pick up how to use the prototype for the customisation feature easily, and were impressed with the result in a short time.
